### Role-Based Weighting & Formula
Weights are applied based on role specialization (Low: 1, Medium: 2, High: 3).

* **Unweighted Score:** `sum(all category scores) / number of categories`
* **Weighted Score:** `sum(score × weight) / sum(weights)`
* **10-Point Equivalent:** `5-point score × 2`
